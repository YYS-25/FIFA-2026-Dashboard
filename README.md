# FIFA 2026 Dashboard

A web-based dashboard for FIFA 2026 tournament data and analytics.

## Setup Instructions

### Local Development

1. Clone or download this dashboard project
2. Open `index.html` in your web browser
3. The dashboard will load and display tournament data

### Google Sheets Export

To use this dashboard with Google Sheets data:

1. Export your FIFA 2026 tournament data from Google Sheets as CSV
2. Convert the CSV data to JSON format
3. Place the JSON file in the project directory
4. Reference it in `app.js` for data loading

### Features

- Real-time tournament standings
- Team statistics and analytics
- Group stage tracking
- Match results visualization
- Interactive filters and search

### File Structure

- `index.html` - Main dashboard HTML template
- `app.js` - Application logic and data handling
- `styles.css` - Dashboard styling
- `utils.js` - Helper functions and utilities

### Browser Compatibility

- Chrome/Edge 90+
- Firefox 88+
- Safari 14+

### Development

Edit the files directly:
- Modify `styles.css` for visual changes
- Update `app.js` for functionality
- Add utilities to `utils.js` as needed

No build process required for basic development.
